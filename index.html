Formularios
<form> Define un formulario HTML 
<form action="submit_form.php">
                        <input type="text">
                        <input type="submit">
                        </form>
 Se usa para recopilar datos del usuario
<input> Define un campo de entrada 
<input type="text">
 Se usa para recibir datos del usuario
<textarea> Define un área de texto 
<textarea>Texto aquí</textarea>
 Se usa para recibir texto largo del usuario
<button> Define un botón 
<button>Haz clic aquí</button>
 Permiten interactuar con el formulario o ejecutar acciones específicas.
<select> Define un menú desplegable 
<select>
    <option value="opcion1">Opción 1</option>
    <option value="opcion2">Opción 2</option>
</select>
 Se usa para seleccionar una opción de una lista
<option> Define una opción en un menú desplegable 
<option value="opcion1">Opción 1</option>
 Se usa dentro de un elemento select
<label> Define una etiqueta para un elemento de formulario 
<label for="name">Nombre:</label>
 Asocia una etiqueta con un campo de entrada
<fieldset> Define un grupo de elementos de formulario 
<fieldset>
    <legend>Grupo de opciones</legend>
    <input type="checkbox"> Opción 1</input>
    <input type="checkbox"> Opción 2</input>
    </fieldset>
 Se usa para agrupar elementos relacionados
Grupo de opciones

<legend> Define un título para un fieldset 
<legend>Texto del título</legend>
 Se usa para describir el contenido de un fieldset
Información de contacto
<optgroup> Define un grupo de opciones dentro de un select 
<optgroup label="Grupo 1">
<option value="opcion1">Opción 1</option>
<option value="opcion2">Opción 2</option>
</optgroup>
 Se usa para agrupar opciones relacionadas
<datalist> Define una lista de opciones predefinidas para un input 
<input list="frutas">
<datalist id="frutas">
<option value="Manzana">
<option value="Naranja">
</datalist>
 Se usa para sugerir opciones al usuario
